策略邏輯 — G6 多週期動能

策略計算

呼叫 run_strategy 輸出目前/下週持倉，形成待買待賣清單。策略計算器 strategy_calculator_g6.py 整合 FinLab 資料。

換股條件

透過 is_rotation_day 判斷 FinLab 交易日曆，只有換股日才執行買賣委託。非換股日僅補單與記錄。

風控整合

警示股/處置股即時檢查（證交所 API + 本地清單），退單自動解析 op_code=88，標記 warning_stocks.json。

程式執行工作流

每日 22:00 由 macOS launchd 觸發 run_daily.sh → main_g6.py --execute

風控機制

警示股偵測

整合證交所處置股 API + 本地 warning_stocks.json，自動標記並跳過警示標的。

退單解析

parse_order_rejection() 解析 op_code=88 等退單原因，自動寫入警示清單。

庫存比對

order_tracker.py 查詢永豐庫存，比對策略應持有，生成補買/補賣清單。

通知模組

utils/notification.py 提供策略報告與下單結果推播，G6 另有 _send_report 即時訊息。

設定參數
